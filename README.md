## calibrateMetaVM

* This repository contains an R package to facilitate calibrating the [`MetaRVM`](https://resume-epi.github.io/MetaRVM/) model to empirical data

### Installation

The package may be installed via `devtools`:

```
devtools::install_github('RESUME-EPI/calibrateMetaRVM')
library(calibrateMetaRVM)
```
